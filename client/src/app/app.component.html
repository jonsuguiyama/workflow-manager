Workflow Manager

System Insights & Architecture Overview

Project Purpose

This application was built to demonstrate full-stack capabilities, modern reactive state management, and robust infrastructure security. It provides a production-ready environment that replaces shared public databases with isolated user sessions to ensure data privacy and automated maintenance.

Architecture & Security Sandbox

Feel free to create, move, edit, or delete tasks to see how the board reacts. Everything you do here is strictly yours and completely invisible to other visitors. When you log in, the system automatically pulls a professional benchmark template I designed and clones it directly into your fresh session, so you don't start with an empty board.

To keep things fresh, a background routine runs automatically every 24 hours, safely wiping temporary sessions and their tasks using an ON DELETE CASCADE database constraint.

Technical Breakdown & Features

  • Reactive State Management: Built entirely with the latest Angular Signals. The application handles real-time updates for asynchronous network operations without unneeded component re-renders, preventing UI glitches.
  • Native Drag & Drop Mechanics: Powered by Angular CDK DragAndDrop modules. Moving a task triggers an optimistic UI update for instant feedback, while the backend recalculates index positions in a batch database transaction.
  • Component & Overlay Architecture: Integrated with Angular Material. Core UI selectors and overlay mechanics like MatSnackBar are customized with global SCSS overrides to deliver unified design feedback without bloating the DOM structure.
  • Responsive UX & User Feedback: Built with a mobile-first approach using flexible CSS Grid layouts optimized for all screen sizes. Non-intrusive toast notifications handle real-time action feedback, while descriptive modal interfaces and native confirmation popups intercept critical events like deletions to prevent accidental data loss.
  • Secure Session Lifecycle: Token verification bypasses localStorage completely to eliminate XSS injection vectors. Session persistence relies on signed JWT payloads delivered via server-side HttpOnly cookies.
  • Route Protection: Routes are guarded using modern Angular functional guards. Protected pages execute a background validation ping to a dedicated verification route (/api/auth/me) before loading any component data into memory.
  • Relational Schema Isolation: Organized with strict database constraints. Tasks and users are decoupled into a clean one-to-many relationship using optimized PostgreSQL raw connection pooling.
